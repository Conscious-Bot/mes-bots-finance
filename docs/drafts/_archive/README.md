d'avant et ont ete refactores en :

| Draft archive | Devenu | Note |
|---|---|---|
| `substack_opening_post.md` (1101 mots EN-only, section pitch finale) | `posts/post_02_comment_that_lied.md` | Compacte ~800 EN, traduit FR, pitch retire (zero pitch principe) |

Les fichiers ici sont des **points de reference historiques** : si on veut
voir comment le draft long contraste avec la version publishable courte,
c'est le diff utile. Ne pas modifier.
